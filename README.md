# Samrat Sharma Furniture (React + Vite)

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```

## Project structure

- `src/App.jsx` - main application UI and state logic
- `src/main.jsx` - React entry point
- `src/styles.css` - Neo-Brutalist styles
- `public/images` - static product assets
